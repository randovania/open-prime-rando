# Open Prime Rando

Open Source randomizer patcher for Prime 2 and eventually 3.

## Standalone patcher

The standalone patcher can apply an Echoes randomizer configuration to an ISO without running Randovania.

```bash
open-prime-rando echoes \
  --input-iso path/to/input.iso \
  --output-iso path/to/output.iso \
  randomizer \
  --input-json path/to/configuration.json
```

The `--input-json` file is the randomizer configuration generated by Randovania. The input ISO is left unchanged and the patched ISO is written to `--output-iso`.

For local development, install the project in editable mode first:

```bash
python -m pip install -e .
```

Then run the same command with `python -m open_prime_rando.cli` if the console script is not on your `PATH`:

```bash
python -m open_prime_rando.cli echoes \
  --input-iso path/to/input.iso \
  --output-iso path/to/output.iso \
  randomizer \
  --input-json path/to/configuration.json
```
